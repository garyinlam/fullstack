# HoloCure DB

Full stack project for final _nology course.

Based off fan game HoloCure, a database for characters, weapons, and items.

Status: In progress...

## Features

- Search database
- Edit database entries
- Delete entries

## Technologies Used

- Java
- SpringBoot
- MySQL
- React.js
